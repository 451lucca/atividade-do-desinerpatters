# Atividades de Design Patterns

Este projeto contem exemplos simples em Java para:

- Strategy
- Observer
- Adapter
- Factory Method

Cada pasta possui seus proprios arquivos `.java` e uma classe `Main` para executar.

## Como executar

Entre na pasta da atividade e rode:

```bash
javac *.java
java Main
```

Exemplo:

```bash
cd Strategy
javac *.java
java Main
```
